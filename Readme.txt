[はじめに]
本アドオンは、1→10design横田による、FlashのビルドインクラスをまねたAS3ライクな記述が
できるopenFrameworks用アドオンです。
各クラスは基本的にパブリックなsetup(), update(), draw()をAPIとして持っており、
プロテクティドな_setup(), _update(), _draw() をオーバーライドする事で
openFrameworks標準の記述と共存できる設計思想となっております。
ofxSelflash/data内にアドオンに必要なアセットが置いてあります。
アドオン使用の際にはこのflashフォルダ毎コピペして配置してください。


[取り扱いについて]
基本的に他人のアドオン、ライブラリは信用しないでください。
本アドオンは「example-UIComponets-1」を参考にデバッグ用の独自UIの為だけに使用する事をおすすめします。

本アドオンは横田がopenFrameworksで複雑なコンテンツを制作する事を目的に作られたものです。
故に基本的には横田のみが使いやすければいいという思想の元に設計されており、AS3を知らない他社が使いやすいとは限りません。

本アドオンをプライベートワーク等で使用するのは問題ありませんが、基本的には1→10design内製のアドオンとして
公開しており、社外への一切の公開を禁止します。
故に、プライベートワークで社外の方との連携で本アドオンが共有される様な使い方はご遠慮ください。
ですが、アドオン内の記述をコピペで使用する分には問題ありません。
